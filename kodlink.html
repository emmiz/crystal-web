Webdesignlänkar

På denna sida kan du hitta nyttiga länkar som kan vara användbara om du funderar på att börja bygga dina egna hemsidor.

  • fonts.com - Om du vill använda udda teckensnitt på hemsidan är det inte säkert att de som sedan skall titta på den sedan har dessa, men det löser du genom att ta hjälp av denna sida.
  • Google Web Fonts - Är ett alternativ till fonts.com.
  • Javascript Kit - Här kan du hitta massor med gratis användbara Javascript.
  • TopUp - På denna sidan kan du hitta ett Javascript som låter bl.a. dina bilder öppnas i ett snyggt popup-fönster. Detta har jag använt på bilderna i min portfolio.
  • w3schools - Här kan du lära dig allt som har med att bygga hemsidor att göra genom att följa olika jättebra tutorials. Sidan är dock på engelska.
  • Webbdesign Direkt - Här kan du hitta en massa intressant att läsa om webbdesign.
  • Webdesignskolan - Även här kan du lära dig att bygga en hemsida med hjälp av tutorials, men denna sida är istället på svenska.

Denna sida är ett webbprojekt i kursen "Webbutveckling - Webbplatsdesign" på Högskolan i Skövde skapad av Emma Jonsson VT 2012.
